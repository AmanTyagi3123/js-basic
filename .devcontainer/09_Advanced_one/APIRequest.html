XMLHttpRequest.readyState property returns the state an XMLHttpRequest client is in. 0 -> UNSENT -> client has been created. open() not called yet. 1 -> OPENED -> open() has been called. 2 -> HEADERS_RECEIVED -> send() has been called, and headers and status are available 3 -> LOADING -> Downloading; responseText holds partial data. 4 -> DONE -> The operation is complete.
